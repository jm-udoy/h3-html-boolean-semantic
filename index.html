[Boolean Attributes of HTML]

Boolean attribute: A Boolean attribute is an attribute that can only be true or false.

How does a Boolean attribute work?

The presence of a boolean attribute on an element represents the “true” value, and the absence of the attribute represents the “false” value.

Yet note:

The values “true” and “false” are not allowed on boolean attributes. To represent a false value, the attribute has to be omitted altogether.

There are 3 types of Boolean Attribute:

  1. name="value"
  2. name=""
  3. name

[Semantic/Non-Semantic Markups]

A semantic element clearly describes its meaning to both the browser and the developer.

Examples of non-semantic elements: "div" and "span" - Tells nothing about its content.

Examples of semantic elements: "form", "table", and "article" - Clearly defines its content.

In HTML there are some semantic elements that can be used to define different parts of a web page:

  • "article"
  • "aside"
  • "details"
  • "figcaption"
  • "figure"
  • "footer"
  • "header"
  • "main"
  • "mark"
  • "nav"
  • "section"
  • "summary"
  • "time"
